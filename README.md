# ReGov Challenge
 
### Run instructions:

After downloading the code, run the following command to start the app:

```sh
$ npm install
$ npm start
```

### The app is built using:

- Typescript
- React
- Redux
- Bootstrap

### Build folder included

run the code below to run build

```sh
serve -s build
```
